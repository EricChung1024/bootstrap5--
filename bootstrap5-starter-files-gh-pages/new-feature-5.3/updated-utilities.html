Object fit

Object fit 的功能是讓圖片或影片在容器中進行填滿、覆蓋、等比例縮放等操作。可以在 .object-fit-{value} 中使用 contain、cover、fill、scale、none。

  • contain：保持原始比例，並完整顯示整個內容，適用於確保完整顯示圖片，而不會被裁切。
  • cover：會填滿整個容器，可能會裁切掉部分內容，適用於背景圖片或視覺設計強調填滿的情境。
  • fill：完全填滿容器，但可能會變形，因為不會保持原始比例。
  • scale：會比較 none 和 contain，選擇較小的尺寸來顯示內容。
  • none：不縮放內容，圖片將按照其原始尺寸呈現。

Object fit 的功能也可以搭配 sm、md、lg、xl、xxl 等中斷點進行使用。

亦可搭配 video 標籤使用。
