Helpers

Color and "Background"

這裡提供了一些可以快速設定文字顏色與背景色的 class，這些 class 會自動根據設定的顏色來決定文字與背景的對比度，讓你不用擔心文字與背景色對比度不足的問題。

更新自 v5.2
基本運用

可以使用各種色彩 utilities，使用 text-bg-{ color } 來套用色彩。

$theme-colors: (
  "primary":    $primary,
  "secondary":  $secondary,
  "success":    $success,
  "info":       $info,
  "warning":    $warning,
  "danger":     $danger,
  "light":      $light,
  "dark":       $dark
);
