Bootstrap Icon 的運用

使用 Bootstrap 官方 icon，連結：https://icons.getbootstrap.com/

方法一：

使用 CND 搭配 ClassName

  • 將 link 中程式碼 加入至 <head> 區塊
  • 使用 bi 作為 ClassName，並搭配 icon 名稱，可參考 https://icons.getbootstrap.com/
Icon link

方法二：使用 Bootstrap Icon Sprite

  • 下載 Bootstrap icon 到本地端
  • 解壓縮至指定路徑
  • 使用 use 搭配 xlink:href 進行套用
  • 嘗試修改 fill 中的配色

優點：不需每次都載入 icon，可減少網頁載入時間
